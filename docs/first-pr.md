# First PR Starter Task

Welcome! Complete this lightweight task to learn the contribution workflow end-to-end.

## Goal
Add a new testimonial quote to the homepage testimonial carousel.

## Steps
1. Create a branch named `docs/first-testimonial` (or similar).
2. Open [`src/components/homepage_components/homepage_testimonial/Homepage_Testimonial.tsx`](../src/components/homepage_components/homepage_testimonial/Homepage_Testimonial.tsx) and locate the `data` array of reviews.
3. Append a new entry that includes:
   - Guest name
   - Short quote (1â€“2 sentences)
   - Star rating value
4. Ensure the testimonial renders properly by running `npm run dev` and visiting the homepage.
5. Update any relevant screenshots if necessary.
6. Run `npm run lint` to confirm lint passes.
7. Commit with a message such as `feat: add Maya testimonial to carousel`.
8. Open a pull request referencing this checklist, add before/after screenshots, and request review.

Completing this task validates your local environment and familiarizes you with the review process.
